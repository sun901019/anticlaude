# Marketing Strategy 複合技能

## 用途

Lala 在進行選題策略、內容規劃、受眾分析時調用此技能。
目標：做出**有數據支撐、有受眾洞察、能轉換的內容策略**。

---

## 一、選題決策框架

### 高品質選題的 4 個條件

| 條件 | 說明 | 檢查方式 |
|------|------|---------|
| **受眾相關性** | 這個主題對 @sunlee._.yabg 的受眾有意義嗎？ | 對照受眾偏好數據 |
| **時機性** | 現在發這個是對的時間點嗎？ | 趨勢、節氣、時事 |
| **差異化** | 這個角度是別人沒有說過的嗎？ | 競品內容掃描 |
| **可行性** | Craft 能在今天寫出高品質版本嗎？ | 是否有足夠素材 |

### 選題評分維度（0-10）
- **話題熱度**：社群上現在有多少人在討論
- **受眾契合度**：和 Sun 的個人品牌定位的匹配程度
- **互動潛力**：能引發留言、分享的可能性
- **轉換潛力**：是否能導向選品或品牌目標

## 二、Threads 內容策略框架

### 內容類型配比（建議每週）
| 類型 | 比例 | 說明 |
|------|------|------|
| 教育型（乾貨、數據）| 40% | 建立專業形象 |
| 故事型（個人觀點）| 30% | 建立情感連結 |
| 互動型（問題、投票）| 20% | 提升互動率 |
| 選品/推薦型 | 10% | 轉換目標 |

### 受眾層次策略
```
認知層（不認識 Sun）
  ↓ 教育型內容、解決痛點 → 轉追蹤
偏好層（有在追蹤）
  ↓ 個人觀點、故事型 → 建立信任
決策層（準備購買/分享）
  ↓ 選品推薦、限時優惠 → 轉換
```

## 三、跨平台策略整合

### 主平台：Threads
- 主力發文，每日 1 篇
- 重點：短文案 + Hook + 互動 CTA

### 延伸策略
- 高互動貼文 → 轉為 IG Reels 腳本（未來）
- 高點閱主題 → 深化為部落格文章（未來）
- 熱門問題 → 開發成 SEO 內容（Sage 接手）

## 四、選題 SOP（每日執行）

### 輸入
1. 今日 Perplexity 抓到的新素材（`articles` 表）
2. 近 14 天高互動主題類型（`posts` 表）
3. 本週品牌方向（`_context/workflow.md`）

### 處理流程
```
1. 掃描素材，找出可延伸的主題
2. 對照受眾偏好數據，過濾低匹配項
3. 進行語意去重（近 7 天發過的主題排除）
4. 評分排序，選出 Top 3 候選
5. 決定主題後，決定「角度」（教育 vs 故事 vs 互動）
6. 交給 Craft 執行
```

### 輸出格式
```
選題：[主題名稱]
角度：[教育型/故事型/互動型]
目標受眾：[誰最有共鳴]
Hook 方向：[第一行大概說什麼]
預期互動：[引發留言/分享/點擊]
素材來源：[文章標題或 article_id]
```
