# Presentation UI — HTML 投影片製作技能

> 來源：frontend-slides（zarazhangrui）
> 本地路徑：`_hub/skills_library/frontend-slides/`

## 適用場景

- 系統架構說明投影片
- 功能展示 Demo 頁面
- 向他人介紹 AntiClaude 系統
- 不想用 PPT 的任何場合

---

## 使用方式

1. 查看原始 skill：`_hub/skills_library/frontend-slides/`
2. 輸出格式：單一 `.html` 檔，瀏覽器直接開啟
3. 投影片範例：`_hub/anticlaude-intro.html`

---

## 設計規範（AntiClaude 風格）

- 深色背景：`#080808` 或 `#0a0a0f`
- 主色：紫色 `#7c3aed`
- 文字：白色 `#ffffff`，次要 `#a1a1aa`
- 字體：Inter（英文）+ Noto Sans TC（中文）
- 玻璃擬態效果：`backdrop-filter: blur(20px)`
